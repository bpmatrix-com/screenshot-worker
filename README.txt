Screenshot Worker v4.4 Full Render Package

Includes:
- Playwright worker
- Full-site crawling
- Same-domain page discovery
- Desktop/tablet/mobile screenshots
- ZIP generation
- GET / and GET /health routes
- POST /capture, /api/crawl, and /crawl routes
- waitMs and postScrollWaitMs support
- basic Cloudflare/challenge detection and wait loop

Deploy:
1. Extract this ZIP
2. Upload the extracted files to your GitHub repo root
3. Redeploy on Render
4. Point the WordPress plugin to your Render URL
